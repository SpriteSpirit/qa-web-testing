Командная строка Windows

Управление файлами и папками

  • copy - Копирование файлов в другое место
  • dir - Отображение файлов и папок в текущем каталоге
  • cd - Переместиться между папками
  • mkdir - Создать папку
  • move - Переместить файлы из одной папки в другую

Приложения и процессы

  • tasklist - Список выполняемых задач
  • taskkill - Остановить или прекратить выполнение задачи
  • (для остановки задачи используется PID, который можно узнать из tasklist).

Системная информация

  • date - Выводит или устанавливает текущую дату
  • time - Выводит или устанавливает системное время
  • host - Отображает имя компьютера
  • systeminfo - Отображает информацию o конфигурации компьютера
  • ver - Позволяет просмотреть версию Windows
