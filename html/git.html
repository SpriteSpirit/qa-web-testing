Git

  • git init - создание нового репозитория Git
  • git clone - клонирование удаленного репозитория с GitHub на компьютер
  • git status - просмотр изменений в репозитории
  • git add filename - добавление отслеживаемого файла
  • git commit -m "commit message" - создание коммита
  • git push - отправка изменений в удаленный репозиторий
  • git log - посмотр список всех коммитов
  • git pull - получение изменений из удаленного репозитория
  • git branch branch_name - cоздание ветки в локальном репозитории
  • git branch checkout branch_name - cоздание ветки в локальном репозитории
